# 고정완의 이력서 (회원인증 백엔드 개발팀 지원)

## 경력

- **Bithumb** 백엔드 개발자 *(2025.09.01 - )*
  - 회원 인증 개발팀에서 근무
  - 회원 데이터 정합성 검증 배치 개발
  - KCB 휴대폰 인증을 통한 SBS 크리스마스 가요대전 방청권 티케팅 시스템 개발 (1.8만 RPS 성능 테스트 통과)
- [**SumOne**](./monymony/README.md) 백엔드 개발자 *(2025.05.12 - 2025.08.14)*
  - DAU 120만명 커플 다이어리 서비스의 회원 데이터의 Hash Sharding을 담당
  - 회원 데이터를 PostgreSQL 에서 DynamoDB 로 무중단 점진적 마이그레이션
  - 변경된 Storage Engine 의 동작원리에 맞추어 키와 스키마 재설계하여 Spatial Locality 확보
    - 커플 위주 서비스이므로 커플 단위로 회원 데이터가 파티셔닝 되도록 키 설계
  - 재설계된 키와 스키마에 따라 DAO 와 쿼리 구현
- [**Dreamfora**](./dreamfora/README.md) 백엔드 테크리드 *(2021.08.25 - 2025.03.25)*
  - 미국, 영국, 인도, 러시아, 이집트 등 전세계 200만명이 사용하는 목표관리 서비스의 백엔드 테크리드를 담당
  - 주요성과
    - IDC에서 AWS 클라우드로 인프라 전환 후 **장애빈도 99%** 감소
    - 쿼리튜닝과 DB 파티셔닝 후 **서버비용 82%** 절약
    - 모놀리식에서 MSA로 리팩토링 후 테스트 커버리지 86% 달성
    - 인증서버를 세션기반에서 토큰기반으로 변경 후 인증시간 93% 단축
- [**Ecube Labs**](./ecubelabs/README.md) 웹 개발자 *(2018.10.15 - 2020.12.28)*
  - 한국 고양시와 협력 하드웨어 업체를 위해 OAuth2.0 인증 서버와 리소스 서버를 구현
  - 일본 이치카와 시의 유료 공공 쓰레기통을 위한 QR 기반 인증 및 과금 시스템을 구축
  - 미국 볼티모어 시의 폐기물 수거 업무 낙찰을 위한 인증 서버 와 E2E 테스트 환경을 구축하고 테스트 코드를 작성
